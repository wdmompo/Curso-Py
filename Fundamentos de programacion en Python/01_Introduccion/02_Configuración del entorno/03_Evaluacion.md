# Preguntas de practica

## Un usuario está configurando su entorno de desarrollo de Python y desea garantizar un proceso de instalación sin problemas con acceso a una amplia gama de paquetes preinstalados de uso común en la Ciencia de datos y el Aprendizaje automático. También desea poder gestionar fácilmente distintos entornos de proyecto y evitar conflictos entre paquetes. ¿Cuál de las siguientes herramientas satisfaría mejor sus necesidades? Seleccione la mejor respuesta.


- [x] Una instalación de Python con la distribución Anaconda
- [ ] Una instalación estándar de Python con gestión manual de paquetes
- [ ] Una instalación de Python utilizando un entorno virtual y la Biblioteca estándar de Python
- [ ] Una instalación de Python utilizando una distribución mínima y pip para la instalación de paquetes

Correcto
Correcto Anaconda ofrece un completo sistema de gestión de paquetes con muchas bibliotecas de ciencia de datos preinstaladas y la herramienta conda para facilitar la gestión del entorno.


## Un científico de datos está trabajando en un proyecto que requiere versiones específicas de varios paquetes de Python. Necesita crear un entorno aislado para evitar conflictos con otros proyectos y garantizar la reproducibilidad. ¿Cuál de las siguientes herramientas sería más eficaz para gestionar este escenario?   Seleccione la mejor respuesta. 


- [ ] Biblioteca estándar de Python
- [ ] Cuadernos Jupyter
- [x] Conda
- [ ] Pip

Correcto
¡Correcto! Conda es un potente administrador de paquetes y entornos que permite crear entornos aislados con versiones de paquetes específicas.

## Un científico de datos está comenzando un nuevo proyecto que implica explorar un gran conjunto de datos, crear visualizaciones y compartir sus hallazgos con colegas en un formato fácilmente comprensible. Necesitan un IDE que facilite la codificación interactiva, admita resultados ricos como tablas y gráficos, y permita la integración de código con texto explicativo y visualizaciones. ¿Cuál de los siguientes IDE se adapta mejor a sus necesidades? Seleccione la mejor respuesta.


- [ ] Un editor de texto básico con un intérprete de Python
- [ ] PyCharm
- [x] Notebook de Jupyter
- [ ] VS Code con extensión Python

Correcto
¡Correcto! Notebook de Jupyter es ideal para la exploración interactiva de datos, la visualización y el intercambio de resultados gracias a su compatibilidad con celdas interactivas, resultados enriquecidos e integración con Markdown.


## Un equipo de científicos de datos está colaborando en un proyecto de Aprendizaje automático. Necesitan una herramienta que les permita compartir fácilmente código, visualizaciones y texto explicativo en un único documento interactivo. También quieren poder experimentar con diferentes fragmentos de código y ver los resultados inmediatamente. ¿Cuál de las siguientes herramientas satisfaría mejor las necesidades de este equipo de Ciencia de datos? Seleccione la mejor respuesta.


- [ ] Un programa de hojas de cálculo como Microsoft Excel
- [x] Notebook de Jupyter
- [ ] Un intérprete estándar de Python en una ventana de terminal
- [ ] Un editor de texto básico con resaltado de sintaxis

Correcto
¡Correcto! Notebook de Jupyter es ideal para proyectos colaborativos de ciencia de datos debido a su capacidad para combinar código, visualizaciones y texto en un entorno interactivo.

## Un científico de datos está explorando un nuevo conjunto de datos y quiere experimentar con diferentes transformaciones y visualizaciones de datos. Necesitan una herramienta que les permita ejecutar código en pequeños fragmentos, ver los resultados inmediatamente y modificar fácilmente el código para iterar en su análisis. ¿Qué característica de Notebook de Jupyter lo hace especialmente adecuado para este tipo de exploración interactiva de datos? Seleccione la mejor respuesta.


- [ ] El soporte de celdas markdown para documentar el código y el análisis.
- [ ] La integración con sistemas de control de versiones como Git.
- [ ] Capacidad para crear presentaciones con código incrustado y visualizaciones.
- [x] La capacidad de organizar el código en celdas que pueden ejecutarse de forma independiente.

Correcto
Correcto La estructura basada en celdas de Notebook de Jupyter permite una rápida iteración y experimentación mediante la ejecución de código en trozos pequeños y manejables.
