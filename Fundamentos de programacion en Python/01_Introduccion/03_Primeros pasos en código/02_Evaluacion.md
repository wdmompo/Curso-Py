## Está desarrollando un programa para seguir el progreso de una campaña de marketing. Selecciona la mejor respuesta.

    
    
- [x] A

    leads_generated = 50
    conversions = 1

- [ ] B 
    int website_visits = 1500;
    int leads_generated = 50;
    int conversions = 10;

> La opcion D es correcta pero no legible para principiantes y puede confundir a otros programadores.
