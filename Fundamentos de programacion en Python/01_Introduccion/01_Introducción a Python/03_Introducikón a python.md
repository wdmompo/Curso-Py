# Python como lenguaje interpretado

Python es un **lenguaje interpretado**, lo que significa que el código se ejecuta **línea por línea** mediante un intérprete.

Esto es diferente de lenguajes **compilados** como:

- C++
- Java

En los lenguajes compilados, todo el código se convierte primero en **código máquina** antes de ejecutarse.

En cambio, Python permite:

- ejecutar código inmediatamente
- ver resultados rápidamente
- experimentar y aprender con mayor facilidad

Esto hace que Python sea ideal para **aprendizaje, pruebas y desarrollo rápido**.

# Programación orientada a objetos (OOP)

Python también admite **programación orientada a objetos (Object-Oriented Programming)**.

Este paradigma organiza el código alrededor de **objetos**.

Un objeto combina:

- **datos**
- **funciones**

La programación orientada a objetos permite:

- reutilizar código
- organizar proyectos grandes
- facilitar el mantenimiento

Se puede comparar con **bloques de construcción**:

cada objeto cumple una función específica y todos juntos forman un sistema completo.

# La belleza de la sintaxis de Python

Una de las características más destacadas de Python es su **sintaxis limpia y fácil de leer**.

A diferencia de otros lenguajes que utilizan muchos símbolos como:

- `;`
- `{ }`
- estructuras complejas

Python utiliza **indentación (espacios o tabulaciones)** para definir la estructura del código.

Esto hace que el código:

- sea más claro
- tenga una estructura consistente
- sea más fácil de entender y mantener.

---

## Ejemplo simple

```python
print("Hello, world!")
```

Una sola línea es suficiente para mostrar un mensaje en pantalla.

Ejemplo con una condición

```python
if grade == "A":
    print("Excellent work!")
else:
    print("Keep studying and you'll improve!")
```

En este ejemplo, la indentación es fundamental.

## Funciones de la indentación

1. Define la estructura del programa

Las líneas indentadas bajo if y else indican qué acciones deben ejecutarse según la condición.

2. Mejora la legibilidad

Incluso sin conocer Python, es posible entender la lógica del código gracias a su estructura visual.

Ejemplo incorrecto sin indentación

```python
if grade == "A":
print("Excellent work!")
else:
print("Keep studying and you'll improve!")
```

Este código:

- es difícil de leer
- generaría un error en Python

Esto ocurre porque Python depende de la indentación para interpretar el código correctamente.

## Tipos de datos: los bloques de construcción

Los tipos de datos son como herramientas en una caja de herramientas.
Cada uno está diseñado para almacenar un tipo específico de información.

Elegir el tipo de dato correcto es esencial para escribir código eficiente y correcto.

- Números

Se utilizan para almacenar valores numéricos.

Ejemplos:

```plaintext
edad → 25

precio → 19.99

número de estudiantes → 30
```

Tipos:

- Enteros (int) → números sin decimales
- Flotantes (float) → números con decimales

- Cadenas (Strings)

  Se utilizan para trabajar con texto.

  Ejemplos:
    - "Alicia"
  -   "Guía del autoestopista galáctico"
    - "¡Bienvenido!"

- Booleanos
    Representan valores lógicos.

  - Valores posibles:
    - True (verdadero)
    False (falso)

- Ejemplo de uso:
    - determinar si una respuesta en un cuestionario es correcta o incorrecta.

- Listas
    Las listas almacenan una colección ordenada de elementos.

    - Ejemplo conceptual:

        Lista de compras:
            
        ```plaintext
        leche
        huevos
        pan
        ```

    - Ejemplo en Python:

    ```python
    shopping_list = ["milk", "eggs", "bread"]
    ```

    - Se utilizan para:
        - guardar colecciones de datos
        - almacenar puntuaciones de un juego
        - mantener listas de nombres. 

- Diccionarios

    Los diccionarios almacenan datos en pares clave–valor.

    Ejemplo:

    ```python
        user = {
        "name": "Alicia",
        "age": 30,
        "email": "alicia@email.com"
    }
    ```

Son útiles para almacenar información estructurada, como perfiles de usuario.
