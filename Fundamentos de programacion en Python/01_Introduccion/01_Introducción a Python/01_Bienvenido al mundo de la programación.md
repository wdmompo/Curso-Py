# Bienvenido al mundo de la programación

## Ideas principales

- La **programación** es el proceso de escribir instrucciones que una computadora puede entender y ejecutar.
- Estas instrucciones se escriben en **lenguajes de programación**, que conectan el pensamiento humano con la ejecución de la máquina.
- Los conceptos básicos incluyen **variables**, **estructuras de datos** y **flujo de control**.
- **Python** es uno de los mejores lenguajes para principiantes debido a su simplicidad, legibilidad y versatilidad.
- Python tiene un amplio ecosistema de **bibliotecas y frameworks** que permiten desarrollar aplicaciones en muchos campos como análisis de datos, desarrollo web e inteligencia artificial.

## Dar instrucciones a las computadoras

La programación consiste en proporcionar a una computadora un conjunto de instrucciones paso a paso para realizar una tarea específica.

Se puede comparar con una **receta de cocina**. En lugar de decir simplemente *"haz un pastel"*, se proporciona:

- una lista de ingredientes  
- cantidades exactas  
- instrucciones claras para prepararlo  

De manera similar, en programación se escribe una serie de órdenes en un lenguaje que la computadora entiende para obtener un resultado específico.

Estas instrucciones incluyen:

- **Lógica**
- **Toma de decisiones**
- **Repetición de procesos**

Un programa puede:

- ejecutar acciones diferentes según ciertas condiciones  
- repetir instrucciones hasta cumplir un objetivo  
- almacenar y manipular datos  

Esto se parece a un **diagrama de flujo**, donde diferentes decisiones llevan a distintos resultados.

---

## Ejemplo de lógica en un programa

Un programa podría calcular descuentos para clientes:

- Si el importe de compra es mayor que **100 $**, aplicar **10 %** de descuento.  
- Si es mayor que **50 $**, aplicar **5 %** de descuento.  
- Si no, **no aplicar descuento**.

Esto se implementa usando estructuras condicionales como:

- `if`
- `else`

## Variables y estructuras de datos

Para trabajar con información, los lenguajes de programación utilizan:

### Variables
Almacenan datos como:

- números
- texto

### Estructuras de datos
Organizan colecciones de datos, por ejemplo:

- **Listas**: secuencias de elementos  
- **Diccionarios**: pares **clave–valor**

---

## Flujo de control

El flujo de control determina el orden en que se ejecutan las instrucciones.

### Bucles
Repiten código varias veces.

- `for`
- `while`

### Condicionales
Permiten ejecutar diferentes acciones según una condición.

- `if`
- `elif`
- `else`

# Lenguajes de programación: las voces de la innovación

Así como las personas usan distintos idiomas para comunicarse, los programadores utilizan diferentes **lenguajes de programación** para comunicarse con las computadoras.

Cada lenguaje tiene:

- **Sintaxis**: reglas para escribir código  
- **Semántica**: significado del código  

Lenguajes populares incluyen:

- **Java**: usado en aplicaciones empresariales.
- **C++**: conocido por su alto rendimiento, utilizado en videojuegos y programación de sistemas.
- **JavaScript**: ampliamente usado en desarrollo web.
- **Python**: famoso por su versatilidad y facilidad para principiantes.

La elección del lenguaje depende de:

- el tipo de aplicación
- la experiencia del programador
- los requisitos del proyecto

Para principiantes, **Python** es una opción ideal por su simplicidad y popularidad.

# ¿Por qué Python?

## ¿Por qué Python? Fundamentos

Python es un **lenguaje de alto nivel** conocido por:

- su **legibilidad**
- su **simplicidad**
- su **facilidad de aprendizaje**

Su sintaxis está diseñada para ser clara y concisa. En lugar de estructuras complejas, utiliza palabras clave similares al inglés.

Esto permite que los principiantes se concentren en:

- la **lógica**
- la **resolución de problemas**

en lugar de en detalles complejos de sintaxis.

## Ejemplo de código

### Python

```python
length = 10
width = 5
area = length * width
print("The area of the rectangle is:", area)
```

### C++

```C++
#include <iostream>
int main() {
  int length = 10;
  int width = 5;
  int area = length * width;
  std::cout << "The area of the rectangle is: " << area << std::endl;
  return 0;
}
```

El código de Python es más corto y fácil de leer, reflejando directamente la fórmula matemática.

## Bibliotecas y frameworks de Python

Python es extremadamente versátil gracias a su gran colección de bibliotecas y frameworks, que permiten realizar tareas complejas sin escribir todo desde cero.

### Análisis de datos

- Pandas: permite trabajar con datos en formato tabular similar a hojas de cálculo.
- NumPy: proporciona herramientas para cálculos numéricos y manejo de matrices.

### Desarrollo web

- Django: framework completo con herramientas integradas para autenticación, bases de datos y rutas.
- Flask: framework ligero que ofrece mayor flexibilidad.

### Inteligencia artificial

- Scikit-learn: incluye algoritmos de aprendizaje automático como clasificación y regresión.
- PyTorch: biblioteca potente para crear modelos de aprendizaje profundo.
