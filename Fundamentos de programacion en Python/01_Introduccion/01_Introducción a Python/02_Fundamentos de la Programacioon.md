# Fundamentos de la programación

## Ideas principales

- La **programación** es el proceso de comunicarse con una computadora mediante instrucciones para que realice tareas.
- Estas instrucciones se escriben en **lenguajes de programación**, como Python.
- Los **fundamentos de la programación** son conceptos básicos que existen en casi todos los lenguajes.
- Estos fundamentos incluyen **variables, tipos de datos, operadores, flujo de control y funciones**.
- La programación ayuda a **resolver problemas dividiéndolos en pasos más pequeños**.
- Está presente en muchos aspectos de la vida diaria, como **aplicaciones, videojuegos, sitios web y vehículos autónomos**.

---

# ¿Qué es la programación?

La forma más simple de definir la programación es como el método mediante el cual las personas **se comunican con las computadoras y les dan instrucciones para realizar tareas**.

Sin embargo, aunque la definición es simple, la programación puede ser bastante compleja.

La programación se puede definir como:

> El arte de escribir instrucciones para que una computadora las ejecute, indicándole **qué hacer, cómo hacerlo y cuándo hacerlo**.

Estas instrucciones se escriben en **lenguajes de programación**, como **Python**.

---

# Fundamentos de la programación

Los fundamentos de la programación son los **conceptos básicos que forman cualquier programa**, sin importar el lenguaje utilizado.

Entre los principales se encuentran:

- Variables  
- Tipos de datos  
- Operadores  
- Sentencias de flujo de control  
- Funciones  

Estos conceptos constituyen los **bloques de construcción de cualquier programa**.

---

# Variables

Las **variables** son contenedores que almacenan datos en la memoria de la computadora.

Permiten:

- asignar nombres a los valores
- reutilizar datos
- manipular información dentro del programa

Esto hace que el código sea:

- más organizado  
- más legible  
- más fácil de modificar

---

# Tipos de datos

Los **tipos de datos** indican qué tipo de información puede almacenar una variable.

Algunos tipos comunes en Python son:

### Enteros (Integers)

Números enteros sin decimales.

Ejemplos:

- `0`
- `10`
- `-5`

### Números de punto flotante (Float)

Números con decimales.

Ejemplos:

- `3.14`
- `-2.5`

### Cadenas (Strings)

Texto entre comillas.

Ejemplo:

- `"Hola"`


### Booleanos (Boolean)

Representan valores lógicos:

- `True` (verdadero)
- `False` (falso)

Usar el tipo de dato correcto es importante para:

- evitar errores
- garantizar que el programa funcione correctamente

---

# Operadores

Los **operadores** son símbolos que realizan operaciones sobre los datos.

Permiten **manipular y transformar la información** dentro de un programa.

### Operadores aritméticos

Realizan operaciones matemáticas:

- suma `+`
- resta `-`
- multiplicación `*`
- división `/`

### Operadores de comparación

Comparan valores:

- mayor que `>`
- menor que `<`
- igual a `==`

### Operadores lógicos

Se usan para combinar condiciones:

- `AND`
- `OR`
- `NOT`

Ejemplo de condición:

> La forma es azul **y** triangular.

---

# Flujo de control

El **flujo de control** determina el orden en que se ejecutan las instrucciones del programa.

Permite:

- tomar decisiones
- repetir acciones
- responder a diferentes situaciones

### Sentencias condicionales

Permiten ejecutar diferentes bloques de código según una condición.

Ejemplos:

- `if`
- `elif`
- `else`

### Bucles

Permiten repetir un bloque de código varias veces.

---

# Funciones

Las **funciones** son bloques de código reutilizables que realizan una tarea específica.

Ventajas:

- organizan el código
- reducen la repetición
- facilitan el mantenimiento

Una función permite **escribir el código una sola vez y reutilizarlo varias veces**.

---

# Programación para resolver problemas

Los fundamentos de la programación ayudan a **dividir problemas grandes en pasos pequeños y manejables**.

Esto permite aplicar un enfoque:

- lógico
- analítico
- estructurado

También ayudan a escribir código:

- eficiente
- optimizado
- rápido
- escalable

---

# Ejemplo de resolución de problemas con programación

Supongamos que debes crear un programa para **analizar las compras de clientes en una tienda en línea**.

El proceso puede dividirse en los siguientes pasos:

1. Obtener los datos de compra del cliente.  
2. Limpiar los datos.  
3. Transformar los datos para analizarlos.  
4. Identificar tendencias o patrones.  
5. Presentar los resultados para apoyar la toma de decisiones.

Durante este proceso se pueden usar:

- **variables** para almacenar datos  
- **bucles** para recorrer conjuntos de datos  
- **condicionales** para tomar decisiones  
- **funciones** para organizar la lógica  
- bibliotecas como **pandas** y **matplotlib** para análisis y visualización
