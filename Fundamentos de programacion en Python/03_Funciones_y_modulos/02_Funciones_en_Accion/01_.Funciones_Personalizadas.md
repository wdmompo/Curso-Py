## 📘 Funciones personalizadas en Python

### 🧩 Concepto clave
- Las **funciones** son bloques de código independientes que realizan tareas específicas.
- Permiten encapsular instrucciones en fragmentos reutilizables, como pequeños programas dentro de tu código.

---

## 🔑 Componentes principales

### 📥 Argumentos
- Son los **valores de entrada** que se pasan a una función.
- Proporcionan los datos necesarios para ejecutar la tarea.
- Permiten que una función se adapte a distintos escenarios.

**Ejemplo:**
- Una función para calcular el salario neto puede recibir:
  - Salario bruto
  - Tasa impositiva
  - Deducciones

---

### 📤 Valores devueltos
- Son los **resultados que la función retorna** tras ejecutarse.
- Se obtienen usando la palabra clave `return`.

**Ejemplo:**
- Una función que valida contraseñas devuelve:
  - `True` si es válida
  - `False` si no cumple los criterios

---

## 🛠️ Cómo crear una función en Python

### Paso 1: Definir la función
- Usar la palabra clave `def`.
- Escribir el nombre (usar guiones bajos si tiene varias palabras).
- Añadir paréntesis y dos puntos.

### Paso 2: Documentación (docstring)
- Describir:
  - Qué hace la función
  - Sus argumentos
  - Su valor de retorno
- Usar comillas triples `""" """`.

### Paso 3: Escribir el código
- Incluir la lógica dentro de un bloque indentado.

### Paso 4: Devolver un valor (opcional)
- Usar `return` si se necesita una salida.

### Paso 5: Guardar la función
- Asegurarse de guardar el archivo para no perder cambios.

---

## ▶️ Llamar a una función

- Una función **no se ejecuta hasta que se llama**.
- Al llamarla, se ejecuta el código definido en ella.
- Permite reutilizar la misma lógica múltiples veces.

---

## 📊 Ejemplo de uso

- Generar reportes de ventas:
  - Una función recibe el período (semanal, mensual, anual).
  - Genera el informe correspondiente según el argumento.

---

## 🚀 Beneficios de usar funciones

- Reutilización de código
- Mayor modularidad
- Código más fácil de entender, depurar y mantener
- Flexibilidad para distintos escenarios
- Mejora en la toma de decisiones gracias a valores devueltos

---

## 🎯 Conclusión

- Las funciones son fundamentales en Python.
- Permiten escribir código **más estructurado, eficiente y adaptable**.
- Dominar su uso es clave para convertirse en un desarrollador competente.
