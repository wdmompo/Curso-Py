- Utilizar nombres de variables muy cortos y comentarios crípticos para ahorrar espacio.
- **Descomponer el código en funciones más pequeñas y reutilizables con nombres y objetivos claros.**

> Correcto El código modular es más fácil de gestionar y depurar.
